\section{Overview}

This document catalogs every figure and table to be produced from
experimental results. Each entry specifies the figure's purpose
in the thesis, its data source, the thesis section it supports,
and its current status. Entries marked \textbf{[LIVE]} include
example outputs from existing Boxing data. Entries marked
\textbf{[PLACEHOLDER]} require completed batch runs.

The catalog is organized by the research question each figure
addresses, following the thesis Results chapter structure
(Chapter~4).

% ================================================================
\section{Part 1: DQN Isolation Study (Thesis 4.1)}
% ================================================================

These figures answer: \textit{Does SPR improve vanilla DQN, and
how does it interact with augmentation?}
Reproduction: Appendix~\ref{app:perf-figs}.

\subsection{A1. Learning Curves -- DQN Isolation [LIVE]}

\noindent\textbf{Purpose:} Show how each DQN condition learns over
the 100K-step budget. The primary performance figure for Part~1.

\noindent\textbf{Format:} 6 subplots (one per game), 4 lines per
subplot (DQN, +Aug, +SPR, +Both). Shaded bands show $\pm 1$ std
from 30 episodes (single seed) or 95\% CI (multi-seed).

\noindent\textbf{Data:} \texttt{evaluations.csv} from each run.

\noindent\textbf{Thesis section:} 4.1 (DQN Isolation Study).

\begin{figure}[H]
\centering
\includegraphics[width=0.82\textwidth]{../output/plots/boxing_learning_curves.png}
\caption{Example: Boxing, seed 42. Full version uses 6-game grid.}
\end{figure}


\subsection{A2. SPR and Augmentation Lift [LIVE]}

\noindent\textbf{Purpose:} Directly visualize the effect of adding
SPR or augmentation. Positive bars mean the intervention helped;
negative means it hurt.

\noindent\textbf{Format:} Paired bar chart. Left panel: score
change from adding SPR (with and without augmentation). Right
panel: score change from adding augmentation (with and without
SPR).

\noindent\textbf{Data:} Final scores from \texttt{evaluations.csv}.

\noindent\textbf{Thesis section:} 4.1.

\begin{figure}[H]
\centering
\includegraphics[width=0.82\textwidth]{../output/plots/boxing_spr_aug_lift.png}
\caption{Example: Boxing, seed 42. Full version groups all 6 games.}
\end{figure}


\subsection{A3. Augmentation $\times$ SPR Interaction [LIVE]}

\noindent\textbf{Purpose:} Show whether augmentation and SPR are
independent, redundant, or interfering. Parallel lines mean
independent effects; crossing lines mean interaction.

\noindent\textbf{Format:} Interaction plot with augmentation on
x-axis, separate lines for with/without SPR.

\noindent\textbf{Data:} Final scores from \texttt{evaluations.csv}.

\noindent\textbf{Thesis section:} 4.1.

\begin{figure}[H]
\centering
\includegraphics[width=0.52\textwidth]{../output/plots/boxing_interaction.png}
\caption{Example: Boxing, seed 42. Full version: one subplot per game.}
\end{figure}

\section{Part 2: Rainbow Capability Comparison (Thesis 4.2)}

These figures answer: \textit{Does SPR need a capable base agent
to produce useful representations?}
Reproduction: Appendix~\ref{app:perf-figs}.

\subsection{B1. Learning Curves -- Rainbow Comparison [PLACEHOLDER]}

\noindent\textbf{Purpose:} Compare SPR's effect on DQN vs Rainbow.
The core capability comparison figure.

\noindent\textbf{Format:} 6 subplots, 3 lines each: DQN+SPR,
Rainbow, Rainbow+SPR.

\noindent\textbf{Data:} \texttt{evaluations.csv} from Rainbow
and Rainbow+SPR runs.

\noindent\textbf{Thesis section:} 4.2.


\subsection{B2. SPR Lift: DQN vs Rainbow [PLACEHOLDER]}

\noindent\textbf{Purpose:} Directly compare the magnitude of
SPR's improvement on a weak agent vs a strong agent, per game.

\noindent\textbf{Format:} Paired bar chart. Per game: SPR lift on
DQN vs SPR lift on Rainbow.

\noindent\textbf{Data:} Final scores from DQN, DQN+SPR, Rainbow,
Rainbow+SPR.

\noindent\textbf{Thesis section:} 4.2.


\subsection{B3. Capability Scatter [PLACEHOLDER]}

\noindent\textbf{Purpose:} Visualize the raw performance gap
between DQN and Rainbow across games.

\noindent\textbf{Format:} Scatter plot. x = DQN score, y = Rainbow
score, one point per game. Diagonal line marks parity.

\noindent\textbf{Data:} Final scores from all conditions.

\noindent\textbf{Thesis section:} 4.2.


\subsection{B4. Full Score Table [PLACEHOLDER]}

\noindent\textbf{Purpose:} Comprehensive results table including
all 6 conditions.

\noindent\textbf{Format:} 6 games (rows) $\times$ 6 conditions
(columns) with multi-seed CI.

\noindent\textbf{Data:} \texttt{evaluations.csv} from all runs.

\noindent\textbf{Thesis section:} 4.2.


\subsection{B5. Human-Normalized Scores [PLACEHOLDER]}

\noindent\textbf{Purpose:} Standardized comparison using
HNS = (agent $-$ random) / (human $-$ random). Standard
Atari-100K reporting format.

\noindent\textbf{Format:} Same as B4, normalized. Requires
published human and random baselines per game.

\noindent\textbf{Data:} Final scores + baselines from Kaiser
et al.\ (2020).

\noindent\textbf{Thesis section:} 4.2.


\subsection{B6. Aggregate Bar Chart [PLACEHOLDER]}

\noindent\textbf{Purpose:} Summary of overall performance across
games. Shows which conditions improve on average.

\noindent\textbf{Format:} Mean HNS across 6 games per condition.
Error bars for cross-game variance.

\noindent\textbf{Data:} Derived from B5.

\noindent\textbf{Thesis section:} 4.2.

\section{Part 3: Interpretability Analysis (Thesis 4.3)}
% ================================================================

These figures answer: \textit{What do the representations encode,
and why do they differ across capability levels?}

All figures in this section require the feature extraction and
probing pipeline (Tasks 47--52). Checkpoints with
\texttt{spr\_state} will be available after batch runs.
Reproduction: Appendix~\ref{app:interp-figs}.

\subsection{C1. Linear Probe Accuracy [PLACEHOLDER]}

\noindent\textbf{Purpose:} Show whether SPR encoders learn more
game-relevant features than vanilla encoders.

\noindent\textbf{Format:} Grouped bar chart. Per game, per
condition: F1 for each probe target (player position, score, etc.).

\noindent\textbf{Data:} Extracted features + RAM labels +
sklearn probes.

\noindent\textbf{Thesis section:} 4.3.


\subsection{C2. Probe Accuracy Over Training [PLACEHOLDER]}

\noindent\textbf{Purpose:} Show how representation quality develops
during training. Do SPR representations improve steadily or plateau
early?

\noindent\textbf{Format:} x = checkpoint step, y = probe F1. Lines
per condition, one plot per game.

\noindent\textbf{Data:} Probes run at each of 10 checkpoints.

\noindent\textbf{Thesis section:} 4.3.


\subsection{C3. Latent Space PCA/t-SNE [PLACEHOLDER]}

\noindent\textbf{Purpose:} Visualize whether SPR produces more
structured representations.

\noindent\textbf{Format:} 2D scatter plots colored by probe target.
Grid: rows = conditions, columns = games.

\noindent\textbf{Data:} Extracted encoder features + labels.

\noindent\textbf{Thesis section:} 4.3.


\subsection{C4. Transition Model Cosine Similarity [PLACEHOLDER]}

\noindent\textbf{Purpose:} Show how accurately the SPR transition
model predicts future states, and whether prediction quality
differs between DQN+SPR and Rainbow+SPR.

\noindent\textbf{Format:} x = prediction step K (1--5), y = cosine
similarity. Lines for DQN+SPR vs Rainbow+SPR.

\noindent\textbf{Data:} Checkpoint \texttt{spr\_state} (transition
model weights). Requires fix from 2026-03-14.

\noindent\textbf{Thesis section:} 4.3.


\subsection{C5. Metric Correlation Heatmap [PLACEHOLDER]}

\noindent\textbf{Purpose:} Identify which Rainbow training metrics
track representation quality. Narrows down components for targeted
ablation.

\noindent\textbf{Format:} Heatmap. Rows = Rainbow metrics, columns
= games, cells = Pearson/Spearman correlation with probe F1.

\noindent\textbf{Data:} \texttt{training\_steps.csv} Rainbow
metrics + probe F1 timeseries.

\noindent\textbf{Thesis section:} 4.3 or 5.1 (Discussion).

\section{Regeneration}
\label{app:regen}

All figures can be reproduced from run data on Google Drive.

\subsection{Downloading run data}
\label{app:download}

\begin{verbatim}
bash scripts/pull-run.sh --all              # all runs
bash scripts/pull-run.sh --group base       # vanilla DQN
bash scripts/pull-run.sh --group spr        # DQN+SPR
bash scripts/pull-run.sh --group rainbow    # Rainbow
bash scripts/pull-run.sh --all --no-checkpoints  # skip .pt files
bash scripts/pull-run.sh --list             # list available
\end{verbatim}

Run directories are documented in
\texttt{experiments/dqn\_atari/runs/INDEX.md}.

\subsection{Performance figures (A, B, D1--D7)}
\label{app:perf-figs}

\begin{verbatim}
python scripts/plot_learning_curves.py  # learning curves
python scripts/plot_results.py          # score tables, bar charts
\end{verbatim}

\noindent Both scripts read \texttt{evaluations.csv} and
\texttt{training\_steps.csv} from local run directories.

\subsection{Post-hoc assessment}
\label{app:reeval}

\begin{verbatim}
python scripts/reeval_checkpoints.py --device cuda  # fast
python scripts/reeval_checkpoints.py --record-video # with video
python scripts/reeval_checkpoints.py --force        # re-run all
\end{verbatim}

\noindent Run on Colab T4, not locally. See
\texttt{reports/eval-pipeline.tex} for details.

\subsection{Interpretability figures (C1--C5)}
\label{app:interp-figs}

Requires Tasks 47--52 infrastructure. Scripts will be:

\begin{verbatim}
python scripts/extract_features.py      # Task 47
python scripts/run_probes.py            # Task 49
python scripts/plot_latent_space.py     # Task 50
python scripts/analyze_transitions.py   # Task 51
python scripts/correlate_metrics.py     # Task 52
\end{verbatim}
