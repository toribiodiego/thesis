## Regeneration Steps

### Complete Workflow

From scratch (after completing training runs):

```bash
# 1. Generate all plots
for run in experiments/dqn_atari/runs/pong_*/; do
    python scripts/plot_results.py \
        --episodes "$run/csv/episodes.csv" \
        --output "results/plots/$(basename $run)"
done

# 2. Export summary table
python scripts/export_results_table.py \
    --runs-dir experiments/dqn_atari/runs \
    --output results/summary/metrics.csv

# 3. Run statistical analysis
python scripts/analyze_results.py \
    --metrics results/summary/metrics.csv \
    --output results/summary/analysis.txt

# 4. Convert to thesis formats
# (Manual or scripted conversion to LaTeX/Markdown)
```

**Est. time**: 5-10 minutes (excluding training)

### Quick Update

After adding new runs:

```bash
# Re-export summary (includes new runs automatically)
python scripts/export_results_table.py \
    --runs-dir experiments/dqn_atari/runs \
    --output results/summary/metrics.csv

# Re-run analysis
python scripts/analyze_results.py \
    --metrics results/summary/metrics.csv \
    --output results/summary/analysis.txt
```

**Est. time**: 1-2 minutes

## Thesis Integration

### Recommended Workflow

1. **Copy artifacts to thesis repo**:
   ```bash
   # Figures
   cp results/plots/pong_42/returns.png ~/thesis/figures/fig1_pong_learning.png

   # Tables
   cp results/summary/metrics.csv ~/thesis/tables/reproduction_results.csv
   ```

2. **Convert tables to LaTeX**:
   ```bash
   # Option 1: Use pandoc
   pandoc results/summary/metrics.md -o ~/thesis/tables/tab3_results.tex

   # Option 2: Manual formatting in LaTeX
   ```

3. **Reference provenance**:
   - Git commit: Include in thesis appendix
   - W&B URLs: Add to bibliography or footnotes
   - System info: Summarize in methods section

4. **Embed figures in thesis**:
   ```latex
   \begin{figure}[htbp]
     \centering
     \includegraphics[width=0.8\textwidth]{figures/fig1_pong_learning.png}
     \caption{Pong learning curve over 10M training frames (seed 42).}
     \label{fig:pong_learning}
   \end{figure}
   ```

## Quality Checklist

Before thesis submission:

### Figures
- [ ] All figures generated at 300 DPI
- [ ] Labels are readable and fonts consistent
- [ ] Captions are descriptive and standalone
- [ ] Legends clearly identify all curves/bars
- [ ] Axes have units and appropriate scales
- [ ] Figures referenced in main text

### Tables
- [ ] All tables have headers and captions
- [ ] Numbers formatted consistently (decimals, significant figures)
- [ ] Statistical measures included (mean ± std or 95% CI)
- [ ] Tables referenced in main text
- [ ] LaTeX formatting correct (if applicable)

### Data
- [ ] Summary metrics match reported values
- [ ] Statistical tests documented and justified
- [ ] Confidence intervals calculated correctly
- [ ] Comparison to paper scores accurate

### Provenance
- [ ] Git commit hash documented
- [ ] System specs summarized in methods
- [ ] W&B run URLs accessible
- [ ] Hyperparameters match paper specifications
