# Documentation

Source code documentation for the DQN Atari-100K codebase.

## Contents

- `reference/` -- Technical documentation for each module
  (models, training loop, replay buffer, environments, config,
  evaluation, logging, SPR, Rainbow)
- `guides/` -- Developer guides (architecture overview, setup,
  testing, workflows, troubleshooting)
- `standards/` -- Engineering and documentation conventions
- `ops/` -- Code quality notes
- `testing/` -- Validation logs

## Scope

This directory documents **how the code works**. Everything else
lives elsewhere:

- Thesis manuscript: `writing/`
- Working results and figure catalogs: `reports/`
- Research papers and notes: `notes/papers/`
- Experiment configs and run data: `experiments/`
- Task planning: `.planning/`
