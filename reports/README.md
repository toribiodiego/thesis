# Reports

Working documents for thesis results and analysis. These are
intermediate artifacts -- drafts, data summaries, and provenance
records -- not the final thesis (which lives in `writing/`).

## Contents

- `working-results.tex` -- Current results draft with figures and
  tables. Compiled with `tectonic working-results.tex`.
- `outline.tex` -- Thesis chapter outline.
- `provenance.md` -- How each figure and table was produced: data
  sources, commands, and dates. Consult this before regenerating
  any result.
